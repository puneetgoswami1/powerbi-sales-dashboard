# Project Summary

## Business Problem
Businesses generate large amounts of sales data every day, but raw data alone does not provide meaningful insights.

This project was created to analyze sales performance, identify trends, and help businesses make better data-driven decisions.

---

## Objectives
- Track monthly sales performance
- Identify top-performing products
- Analyze regional sales trends
- Monitor profit growth
- Build an interactive dashboard

---

## Dashboard Features
- KPI cards
- Monthly sales trends
- Profit analysis
- Region-wise filtering
- Interactive visuals

---

## Technologies Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization

---

## Key Learnings
Through this project, I improved my skills in:
- Dashboard design
- Business analytics
- Data storytelling
- Visualization techniques
- Data interpretation
